## جدول الخدمات — الرياض 2026

| الخدمة | يشمل | لا يشمل افتراضياً |
|--------|------|-------------------|
| كشف تسربات عام | مسح حراري + صوتي + تقرير مصور | تكسير تشطيبات |
| كشف خزان علوي | فحص overflow + خط تغذية + ضغط | إعادة عزل خارجي كامل |
| كشف بدون تكسير | تحديد مربع 30×30 سم قبل الفتح | إعادة بلاط كامل |
| عزل أسطح | فوم / بيتومين حسب التقرير | إصلاح تشققات خرسانة عميقة |
| عزل خزانات | إيبوكسي داخلي | استبدال خزان |
| تسليك مجاري | فحص + تسليك خط رئيسي | حفر خارجي |

## مناطق الاستجابة في الرياض

نغطي أحياء شمال ووسط وشرق الرياض (النرجس، الياسمين، الملقا، العليا، النخيل، قرطبة، وغيرها). **موعد الزيارة** يُحدد حسب موقعك — المقر الرئيسي في جدة مع فرق ميدانية للرياض.

## خطوات الحجز

1. [اتصل أو واتساب](/contact) — صف العرض (فيلا/شقة/عمارة).
2. معاينة أولية — تحديد نطاق الفحص.
3. تقرير PDF — موقع التسرب + تكلفة إصلاح تقديرية.
4. تنفيذ — كشف ثم إصلاح/عزل حسب الموافقة.
