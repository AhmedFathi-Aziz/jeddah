# أخبار الموقع (Markdown)

**كل ملف `.md` هنا = صفحة خبر مستقلة** على `/news/[slug]`

## إضافة خبر جديد

1. أنشئ ملفاً: `content/news/اسم-الخبر.md`
2. املأ الحقول بين `---` (بما فيها **المصدر**: اسم الموقع + الرابط)
3. اكتب **نص الخبر بصيغة Markdown** تحت `---` (انظر القسم التالي)
4. احفظ ثم `npm run dev` أو `npm run build`

## نص الخبر (Markdown)

كل ما تحت `---` الثانية يُعرض في صفحة الخبر كـ **Markdown** (عناوين، قوائم، روابط، غامق، …) — مثل المدونة.

> **مهم:** العنوان الرئيسي من حقل `title` في الأعلى — لا تكرّره بـ `# عنوان` في النص (يسبب تكراراً ومشاكل عرض). استخدم `##` للعناوين الفرعية فقط.

### أمثلة سريعة

```md
## عنوان فرعي

فقرة عادية مع **نص غامق** و[رابط لخدمة](/leak-detection).

- نقطة أولى
- نقطة ثانية

> اقتباس أو تنبيه مهم

1. خطوة واحدة
2. خطوة ثانية
```

| الصيغة | Markdown |
|--------|----------|
| عنوان فرعي | `## عنوان` |
| غامق | `**نص**` |
| رابط داخلي | `[النص](/contact)` |
| رابط خارجي | `[النص](https://example.com)` |
| قائمة نقطية | `- عنصر` |
| قائمة مرقمة | `1. عنصر` |
| اقتباس | `> نص` |

## الحقول (YAML)

| الحقل | مطلوب | الوصف |
|--------|--------|--------|
| `slug` | نعم | معرّف إنجليزي فريد للرابط |
| `title` | نعم | عنوان الخبر |
| `excerpt` | نعم | ملخص في قائمة الأخبار |
| `publishedAt` | نعم | `2026-05-22` |
| `source` | يُفضّل | **اسم الموقع** + **رابط الخبر الأصلي** (انظر الأسفل) |
| `published` | لا | `false` لإخفاء الخبر |

### المصدر (اسم الموقع + الرابط)

```yaml
source:
  name: سبق
  url: https://sabq.org/example-article
```

- `name` = اسم الموقع أو الجهة (سبق، واس، وزارة البيئة، …)
- `url` = رابط الخبر الأصلي كاملاً

بديلاً يمكن كتابتهما منفصلين: `sourceName` و `sourceUrl` (نفس المعنى).

يظهر المصدر في **قائمة الأخبار** وفي **صفحة الخبر**.

## مثال كامل

```md
---
slug: saudi-water-campaign
title: مبادرة وطنية لتوفير المياه
excerpt: توعية بأهمية كشف التسربات في المنازل.
publishedAt: 2026-05-22
source:
  name: واس
  url: https://www.spa.gov.sa/example
---

## ملخص الخبر

الفقرة الأولى عن الخبر وربطه بخدماتكم في المملكة.

- [كشف التسربات](/leak-detection)
- [تواصل معنا](/contact)

**تنبيه:** ارتفاع الفاتورة قد يدل على تسرب خفي.
```

## أين يظهر؟

- القائمة: `/news` (مع سطر المصدر إن وُجد)
- صفحة الخبر: `/news/slug` (عنوان + نص + رابط المصدر)
